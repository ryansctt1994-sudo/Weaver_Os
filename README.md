# Weaver OS

**Deterministic verification and replay substrate for bounded probabilistic cognition.**

A verifiable execution system that proves rejected authority claims cannot mutate state.

## Core Thesis

Weaver OS provides a replay-verifiable constitutional execution substrate for bounded AI agents. By maintaining deterministic execution traces and cryptographic receipts, it enables:

- **Proof of rejection**: Demonstrate that rejected authority claims produced no state mutation
- **Deterministic replay**: Re-execute computations with identical outcomes
- **Constitutional boundaries**: Enforce policy constraints at the execution layer
- **Verifiable cognition**: Create auditable records of agent decision-making

## Public Milestone: The Verifiable Spine

This MVP demonstrates a deterministic, replayable execution spine that proves rejected authority claims cannot mutate state.

### Directory Structure

```
canonical/     # Constitutional policies and execution rules
chronicle/     # Deterministic execution trace logging
policy/        # Authority claim validation and enforcement
kernel/        # Core execution engine with replay capability
receipt/       # Cryptographic proof generation and verification
```
